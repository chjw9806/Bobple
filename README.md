# 주요 기능


- **위치 및 검색 데이터 기반 맛집 추천**
    - 사용자 위치 근처 맛집 지도 제공
    - 실시간 검색 데이터를 기반으로 한 메뉴 추천
- **유저 간 정보 공유**
    - 도시락 레시피 공유
    - AI 기반 재료 맞춤형 레시피 추천
- **사용자 소통 및 모임**
    - 함께 식사할 사람 모집 및 채팅
    - 함께 식사 후 영수증 인식을 통한 정산 기능
- **재미와 포인트**
    - 음식 맟주기, 룰렛, 가챠 등 다양한 게임 제공
    - 게임으로 얻은 포인트로 기프티콘 구매 및 사용
