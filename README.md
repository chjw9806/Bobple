## 🪄구현 기능


- **채팅방 생성 및 관리 시스템**
    
- **소켓을 활용한 실시간 채팅기능 구현 / 메시지 읽음 처리 및 읽지 않은 메시지 수 제공**

- **참여자 관리 및 역할 기반 접근 제어 (RBAC)**
  
- **SSE를 기반으로 한 실시간 알림 스트리밍 구현**
  
- **알림 API 설계 (CRUD 및 상태 관리)**
